Commandeer

[eugene-eeo/commandeer]

Commandeer is a little Python DSL for doing beautiful commands and building apps that leverage the command-line. 

from commandeer import *
git = Command('git')

Underneath the hood commandeer uses the envoy library to run the commands and return the responses, while only generating commands from method calls. 

pipe = git.log(n=20) | grep('[a-f0-9]\{40\}', o=None)
response = pipe.run()

If you are interested in contributing or using the library take note that it's API is still in beta mode and breaking changes can always be made.
